HTML Tables

Tables were used to layout websites, by positioning content across the webpages.

Now however, tables should be reserved for tabular data now.

Few Elements:

  • td: Single cell in a table that contains data.
  • tr: used for a row.
  • th: Used for the single cells of a header

The following add merge to the table:

  • colspan: to add merge to the coloums
  • rowspan: to add merge to the rows

The following breakup the table into sections (good for semantics):

  • thead: for headers, especially if we have more than one row of headers (in case of merged cells).
  • tbody: for all the data rows
  • tbody: for footers when we have things like total etc.
