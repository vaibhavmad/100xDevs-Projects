HTML is a markup language, which adds markup to various elements on web pages. Markup means, using various symbols or special characters to highlight elements and provide these elements with a structure.

These structures can be headings, paragraphs, lists, etc.

HTML contains all the sturcture of a web page, without the interactions or styling.

Simple / Basic Elements in HTML

  1. Headings: Range from H1 to H6. Only one H1 per page. Used for SEO as well.
  2. Paragraph: To contain paragraphs in separate text boxes. Syntax is < p > and < /p >.
  3. Bold: Used to make a text bold. Uses b tag: < b > text in between < /b >. Here's a sample: This is a sample.
  4. Italics: Used to italisize text. Such as This is a sample.
  5. Lists: There are two types of lists in HTML, Ordered and Unordered.
    • This text is stored in an unordered list.
    • The unordered list is a list without numbering and uses certain symbols to separate list items from another.
    • Below is a sample of an ordered list. Simply a list with numbered elements.
      1. Element 1
      2. Element 2
      3. Element 3
  6. Anchor Tag: An anchor tag is used to add hyperlinks to a specific text, which when clicked takes the user to the added link destination. For example: google, shall take you to google's website. Here's one more. Click here.
  7. Images: As the name suggests, simply a syntax to add images to the pages.

    As shown, the above image is being fetched and is being added to our page.

    • Image has only one tag. No closing tag.
    • There are certain attributes in the image that must be added to the image. One such is alt, or an alternate text for screen readers.
  8. Comments: Add information in the code, that should not go in the production code, just for information that needs to passed on for internal code reference. Syntax is < !-- comment -- >

Sidenote on HTML Boilerplate: This is essential structure that is needed in an HTML page. These include the doctype, HTML tag, head tag and finally the body tag.

  • Both Head and Body are inside the HTML tag.
  • Head contains the metadata of the page. The metadata elements are like: title of the page, scripts, stylesheets references etc.
  • Title element is used by google to show the page on the search results page and also it is visible in the browser tabs. It should be on every single page that we create.
  • Body contains the content that is visible to the user. Such as text, images, buttons etc.
  • Doctype declaration tells the browser that this page is build on HTML.
