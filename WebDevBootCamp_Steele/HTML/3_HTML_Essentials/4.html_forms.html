HTML Forms

Form is a sort of a shell or a container, under which multiple elements are grouped under the 'Form' tags. These are individual input and button elements.

When we create form tags, we also get an attribute by the name of action - which primarily deals with where the data should be sent.

So when a form is submitted, a HTTP request is sent with the information that we are searching for and this information is then searched on the server and finally we get the result.

Form Elements

  1. Input Elements: This element has no closing tag and has an attribute - type (which specifies how the input will behave) and placeholder (basically the text that is mentioned before user enters information).

    Input Types

    1. text: as the name specifies, takes text as input.
    2. password: exactly as above, however, hides the text being written.
    3. color: gives us a color picker.
    4. file: gives option to upload a file.
    5. number: takes a number as an input from a dropdown (depends upon browser).
    6. time: time is taken as an input.

    Note:

    1. There are a lot of input types and the above list is not exhaustive. There are other such as range, etc. that shall be covered later.
    2. Placeholders does not work for a few elements such as date, time, colorpicker etc. This is because there is no option to show text due to the structure of these elements or due to the design of these elements.

  2. Label: Text that defines the element details and is attached to the input section. If a user clicks on a label of the attached input, the input box becomes active. This is also good for the screen readers.

    For label to work correctly, we need to mention for attribute in the label tag and then that same for attribute should be added as a id attribute in the input tag.
