What is HTML5?

HTML5 is the latest evolution of the standard that defines HTML. New elements and behaviours have been added to HTML5. Ideally it is HTML5 and More components, however, for short people call it HTML5.

It has new stuff, that is not only HTML5 but supports other languages as well. HTML simply defines how HTML works and the latest version of that HTML instructions is HTML5. HTML specs can be viewed by clicking here .

So, browser companies develop browsers on the basis of these specs. There is nothing that we need to update.

Intro to Semantic Markup

This is a recent trend. Very simple to understand.

The term semantic means something related to meaning. The semantic markups are used to show the meaning of code.

Some examples are < nav >, < main >, < article >, < section >, < header >, < footer > etc..

Old semantic markup is used in Medium.com. To check the new sematics being used, we can check stripe.com.

It matters since it tells search engines like google about the content of the page and helps in better structure explanation. It also helps in navigating the code foe us and for other developers.

Here's a list of some of these elements:

  • Main: Main content of the page, sans nav, foooter, header etc.
  • Nav: As the name suggests to add navigation links in one section.
  • Section: To specify a section other within the main tag.
  • Article: Supposed to be a self contained composition.
  • Aside: Something that is tangential, sidebars, callout boxes. Not the primary content.
  • Headers and Footers: Introduction content is header and end content is footer.
  • Then there are certain other elements such as figure, abbr (abbreviation), data, etc..
