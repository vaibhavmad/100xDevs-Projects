CSS Selectors

Selecting by an element is something that we have already covered in the previous segment. However, there are a specific set of selectors that help us in style only specific elememts, example: only a couple of buttons on a page containing half a dozen buttons.

*: Star is also known as an universal selector, hence covers every element in HTML. It is not a good idea usually to use this to use styles.

We can also use a comma to combine multiple selectors, such as h1, h1 {} can be used to style h1 and h2 simultaneously.

Important & Most commonly used Selectors:

  1. ID Selector: We had earlier used ID to combine inputs with labels. Similarly, we can use ID to select a specific element in CSS. ID is used when we have to style a specific single element. Pseudo Classes

Keywords that we add to the selector that modify and specify a particular state of a particular elements. Such as a check box that has been checked or if we are hovering on top of an anchor tag.

There are multiple pseudo classes, here, however we shall cover the most common ones. Point to be noted is that pseudo classes start with a colon(:), so signify that we are using a pseudo class. The most common ones are:

  1. : hover: Hover is going to modify our selection, when the cursor is hovering over an element. To show how it works, lets create few buttons below under a section with class hover and a few buttons outside the section.







  2. : active: When the button is active. Or is clicked. Click on above button 3 outside.
  3. : checked: When radio or checkboxes are checked.
  4. : nth-of-type: if there are multiple elements of the same type and we want to something to every second or every fifth, this is what we need to use. Here an attribute has to be added element:nth-of-type(3) shall get the third of the element, however, if we use element:nth-of-type(3n) shall add styles to every third. For more pseudo classes, check this page .
