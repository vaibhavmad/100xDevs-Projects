CSS Selectors

Selecting by an element is something that we have already covered in the previous segment. However, there are a specific set of selectors that help us in style only specific elememts, example: only a couple of buttons on a page containing half a dozen buttons.

*: Star is also known as an universal selector, hence covers every element in HTML. It is not a good idea usually to use this to use styles.

We can also use a comma to combine multiple selectors, such as h1, h1 {} can be used to style h1 and h2 simultaneously.

Important & Most commonly used Selectors:

  1. ID Selector: We had earlier used ID to combine inputs with labels. Similarly, we can use ID to select a specific element in CSS. ID is used when we have to style a specific single element. Such as a single button has to be distinguished and given a different background color. Only one thing on a page should have only one ID. Below's an example:

  2. Class Selector: It is similar idea as ID, however, the only difference is that it can be applied to multiple elements. For example, if there is a repeated pattern, such as on a social media webpages, there are multiple H3's including the profile names on the wall and other headings. Now, to separate, if we want to style the profile names separately, we can use class. Instead of using #idname, we use .classname as the selector in css.

So, if we only want to style the anchor tags that are direct children of the section, we can use 'section > a {}' as the selector.
