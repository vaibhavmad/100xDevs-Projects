CSS Box Model

The box model is just an idea that everything in CSS is a box with certain properties. Even an icon or a circle is contained as if it is present inside a box. The code pieces of a CSS box are:

  1. Inner Content Box
  2. Padding
  3. Border
  4. Margin

The outer most layer is margin, followed by border, which further contains padding and finally we have Inner Content box.

2. Border & Border Radius

There are many border related properties. Borders play an important role in bringing attention to some elements. The main properties are: border-width, border-color & border-style. These are the basic properties of a border and all three are needed in order to show the border. Below are examples of different border types. All three values can also be provided in one parameter [border: medium dashed green;] {Box 5 below}.

Box 1
border-color: black;
border-width: 5px;
border-style: solid;
width: 200px;
height: 200px;
Box 2 NO BORDER background-color: lightblue; width: 200px; height: 200px;
Box 3
border-color: black;
border-width: 5px;
border-style: solid;
width: 190px;
height: 190px;
Box 4 border-color: green;
border-width: 10px;
border-style:double;
width: 200px;
height: 200px;
box-sizing: border-box
Box 5 border: 10px green groove;
width: 200px;
height: 200px;

Note: In the above example we can see that even though the width and height of both the boxes is same, still Box 1 is bigger than Box2. So, here the key point to be observed is that border width, has been added over and above the dimensions of the inner box. Therefore, box 3, which has an inner content dimensions of 190px X 190px, is equally sized to the non border box, as the overall dimensions are same.

This point clearly shows that border dimensions are over and above the inner box dimensions. However, the same can be corrected, meaning the border and the innercontent can be given one dimension, instead of two, by adding another property - [box-sizing] and giving it the value of [border-box]. This can be seen in the fourth box. Here, we have border width of 20px and box width of 200px, however, the total box width is also 200px and not 220px.

Note: Also note that while we have applied these properties to all sides, these properties can be individually decalred for each side specifically as well.

3. Padding

The space between the inner content and the border. Can be added to all sides at once or can be done separately for each side depending upon the requirements. Below are some examples:

No Padding
padding: 20px
padding-left: 20px
padding-bottom: 20px

Note: We have a shorthand to set padding separately for all 4 sides as well, it is simple: padding: 20px(top) 20px(right) 20px(bottom) 20px(left) or padding: 20px(top and bottom) 20px(right and left).

4. Margin

Same as the padding, all same shorthand and properties, however, only difference is that is works for setting up the space outside the border and other elements, while padding is inside the border sandwiched between border and inner content.
