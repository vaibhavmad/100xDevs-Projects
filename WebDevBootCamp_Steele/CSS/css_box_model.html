CSS Box Model

The box model is just an idea that everything in CSS is a box with certain properties. Even an icon or a circle is contained as if it is present inside a box. The code pieces of a CSS box are:

  1. Inner Content Box
  2. Padding
  3. Border
  4. Margin

The outer most layer is margin, followed by border, which further contains padding and finally we have Inner Content box.
