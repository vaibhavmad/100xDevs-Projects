Opacity & Alpha Channel

  1. Alpha Channel: Alpha channel deals with transparency. We have read about rgb color scheme. In rgba we add another element 'a' and make it rgba. Now, a here stands for alpha channel. It's value is from 0 to 1. O being transparent and 1 being zero transparency.

    We can see an example below. We have two divs containing one div each. The first parent div has a non transparent internal div, while in the second case, the alpha channel is set to 0.7 for the inner div.

    Note: An important point to be noted here is that in case of alpha channel, only a specific element's transparency is affected. If we add any text inside the partially transparent div or any other element, in that case, the text will have 100% transparency. Below is an example of the same:

    This is test text.

  2. Opacity: Opacity as the name suggests is very similar to alpha channel as it controls tha opacity of an element. However, there are some fundamental differences between opacity and alpha channel.
    1. When we talk about opacity, it is a separate property and ranges between 0 to 100 when it comes to the value.
    2. Also, Unlike alpha channel, opacity when declared for an element, gets applied to the children as well.

The Position Property

The position element is used to set the position of elements on a webpage. It is needed when we have to design certain elements or multi layered elements. The most common properties are:

  1. Static
  2. Relative
  3. Absolute
  4. Fixed
  5. Sticky

These properties are used to place elements on the page. Learn this with practice as this is a huge topic and not needed for basics at this moment.
