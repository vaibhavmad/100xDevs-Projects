What is CSS?

CSS also known as cascading style sheet is used to add style or beautification to the applications or controls the look and feel of the same. HTML is added first and then the style is added.

Note: CSS is easy to get the hang of, however, can be intimidating since CSS is huge.

Therefore, the mindset should be not to know all the elememts. Practice to remember most common ones and research for complicated ones.

How to add styles in HTML:

  • Add Styles in HTML Tags: Aware of this, hence not going into much detail. This is not a good practice as this basically requires styling each element separately.
  • Add Style Element in Head: Aware of this, hence not going into much detail. This is not a good practice as this basically requires styling each HTML file separately.
  • Add Elements in a separate CSS file: best can be used for multiple pages and multiple elements in a single page. Name can be filename.css (example: style.css).

Adding a stylesheet into HTML: To do this, we need to use 'link' tag in the head of the HTML file. This tag contains two attributes:

  • rel: Means relation, where we need to specify the relation of the added link with the HTML file. For CSS links, we add "stylesheet" as the relation.
  • href: As the name suggests, this is where we add the file address / reference.

Here's an example of a 'link' tag:

<link rel = "stylesheet" href& = "styles.css">

CSS Rules

Almost everything in the CSS follows the below pattern:

element(that needs to be styled) {
       property(such as height, width, color): value(value of the property);
}

Below are a few examples of the same:

  • h1 {
           color: purple;
    }
  • img {
           height: 100px;
            width: 100px;
    }

Basic CSS Properties:

  1. Color: Color's property syntax is 'color'. Multiple color systems such as RGB or HEX can be used to specify the value. This property is used to manage the color of text.

    For example, all paragraphs of this page are in dark blue color.

  2. background color: As the name suggests, changes the background color. Below example buttons, we are using both color (to change text color) and background color.

    Note: The background color shall fill the entire block of the element. Example is this section only.
