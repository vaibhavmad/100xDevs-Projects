1. What is Bootstrap?

Bootstrap is framework(contains the basic design elements, that can be imported into a website and further enhanced as per the requirement), and contains pre-written code for HTML, CSS & JS. It helps in quickly creating a website using pre-existing layouts(templates) and components(such as navigation, gallery, etc.). As per Colt, we should not always rely on tools like Bootstrap, however, we should be aware of the fact that these exist.

The most important aspect of Bootstrap is Grid which adds the responsive features, and we shall cover this later. Most common websites use similar frameworks to quickly deploy responsive sites. To access bootstrap website, click here.

2. Including Bootstrap & Containers

2.1 Including Bootstrap

To add Bootstrap CSS, we have two options. The first one is downloading the entire CSS code, save it locally, like we do with CSS files and then link them in the HTML code. The second method is to pick up a pre-built link from Bootstrap's website and then link it in the document. The styling is then sent over the network via a CDN.

Note (Important):

  • As we have mentioned above that Bootstrap contains frontend code(HTML, CSS, JS) and some of the functions can not work without JS. Now, we have not covered the same so far. However, for Bootstrap to work properly, we shall have adjoining JS in the file. Now, the way to add JS is similar to adding CSS. We need to add script using script tag. Also, the same is added at the bottom, to help with page speeds.
  • In case of production application, we shall download these files instead of linking in this way. The links can be changed or the site might go down, hence.

We have added both CSS and JS linked to this file as well and shall style it using Bootstrap alone while we learn the same.

Now before we move ahead, let us understand how Bootstrap works. The link that we have added for stylesheet above, is a very similar stylesheet to what we create locally. It has a bunch of classes, containing various properties and corresponsing values. Now, to add a pre-defined bootstrap style to an element on our webpage, we need to check on bootstrap website for the class and add that as a class to our element. As soon as the class is added, the corresponding style gets applied to that element. For example: we have class = "container" applied to the main tag on this page. On the basis of that alone, we have the margins and padding applied to the same.

2.2 Bootstrap Containers

bootstrap containers are the classes that help in defining the overall structure of a page. For example:

  • class: container: Simple single container that has 3 break points for mobile, laptop and tablet. The content becomes responsive on the basis of the viewport(screen) size.
  • class: container-fluid: Spans the entire width of the viewport.

    • There are other containers as well such as container-md(medium), container-sm(small) etc.
