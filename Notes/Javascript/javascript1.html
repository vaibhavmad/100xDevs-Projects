Javascript

1. Introduction

Javascript is a programming language and is one of the three essential front-end technologies alongside HTML & CSS. We now know that HTML is esential for adding the elements on a web page and CSS is used for it's styling. By adding javascript, we make the web-page interactive. For example, we have a calculator on a page. To add buttons containing synmbols and numbers, we use HTML. To make it look like a calculator we use CSS and finally to make the calculator work, for what's its intended we use Javascript.

Javascript, while being versatile, is also a great choice in terms of a programming language because of the available frameworks, that enable using JS for the entire tech stach we need for a fully working website.

In the following days, we shall cover Javascript from the basics and then further proceed to more advanced functionalities.

2. JS Basics

Before we get into how JS works with HTML & CSS, we need to learn how JS actually works by going through the programming basics. To start this, we shall first work with the primitive types.

2.1 Primitive Types

Primitive types in JS are the types of data /information that JS can store and handle. The primitive data types available in JS are:

  1. Numbers
  2. Strings
  3. Boolean
  4. Null
  5. Undefined

There are two more primitive types: Symbol and BigInt. However, we shall not cover these as of now as these are not very common.

Now let's start with numbers:

  1. Numbers: In JS, there is one number type(unlike other languages where we have float, int, etc.). Numbers can be positive, negative, ints and floats. We can do math operations using JS. This is performed by using operators, these are the common ones:
    1. Addition > +
    2. Subtraction > -
    3. Multiplication > *
    4. Division > /
    5. Modulo > %
    6. Exponent > **

    NaN: We will come across a concept in JS, known as NaN or Not a Number. Things like 0/0 or 1 + NaN are given as NaN in JS. Below in an example of the same:

    Math Object

    Math object is an inbuilt collection of matahematical properties, that helps in performing math operations. These are done by using math.properties(). Some examples are:

    • Math.floor(23.9901) shall give us 23. Simply chop of decimals.
    • Math.random() shall give us any random decimal between 0 and 1. Further calculations with this are used to generate random numbers between given set of inputs. Refer last video of strings for this.

  2. Strings: Strings are the data type related to text or textual information (strings of characters). Strings are always written in " " (inverted quotes). We can also use single quotes, depends on the developer, however, in one file we should remain consistent and shall have only one type of quotes sorrounding the text.
    Important

    > Strings are indexed, meaning each character has a corresponding index. The index start with 0 and goes on till the last number. For example:

    H E L L O
    0 1 2 3 4

    As we can see above, the index always starts with 0 and ends at the last character. Individual charactes can be access by using [index] after the string. For example:
    HELLO[3] will be L
    or
    Animal[0] will be A

    Important

    > Also, all strings have a built in property, which is length. We can simple do
    Animal.length will be 6
    Here, length will give us the number of characters and not the highest index. It is to be noted that length also accounts for spaces and other characters in a string.

    Important

    Strings can be combined together using concatenation by using '+' sign. If we have first name and last name of a person and we add both using +, the first string will be prefixed to the second one as in the example below:

    "Virat" + "Kohli" will give us "ViratKohli".

    Also, if we concatenate a string with a number, both types are treated as strings in this case. We shall learn more about it in future and here's an example of the same below

    "Virat" + 18 will be "Virat18"

    3. ****String Methods****

    String methods are built-in actions that help us in manipulating indiviual strings. Some examples of these actions are: searching within a string, replacing part of a string, chaging from lowerCase to upperCase. The syntax for using a method is similar to using length(which is a property): hello.method(). We need to add a method and must add brackets at the end of it.

    The most common strings methods are:

    • .toUpperCase()
    • .toLowerCase()
    • .trim(): It will remove any white space at the beginning and end of a string.
    Methods can be combined in a single string to perform any actions. For example we can have hello.trim().toUpperCase() .

    So far we have used methods with brackets with nothing inside them. However, there are certain methods where we can add values within the brackets. This value that we pass in the brackets is known as arguments. Some of these are:

    • indexOf(): Pass a character in the brackets, such as in case of hello.indexof('h') and we shall get 0 as the answer. It must be noted, that we will only get the first occurence of the character, in case of duplicate characters. For example, in case of hello.indexOf('l'), the answer will be 2 as the first 'l' is at index 3. Also, if we pass in an argument that is not present in the string, we shall get -1 as the response. We can also use longer strings and not charactes only, such as hello.indexOf('ell') shall give us 2 as the answer.
    • slice()
    • replace()
    • repeat()

Variables are labels for Data, so as to declare the same once, manipulate and refer back to the same again. Variables are used to store information under labels, such as name, numbers, etc..

To declare a variable, we use the following syntax:

let variableName = value;

An example of the same is:
let year = 1988;

Variables are used to store values and then perform operations on those values. For example:

Everytime we need to add any value to a variable, we follow the below syntax:
let score = 0;
score = score + 5
score = score + 5

Here, we have added 5 to the score twice and the final value of score is 10. However, there is a shorter way to do this. Therefore, instead of doing the following
let score = 0;
score += 5;
score += 5;

This output of the syntax above is same as the earlier longer syntax. Similar to addition we can do subtraction also, by using score-= or multiplication by using score *= 10.
Using this type of calculation for adding 1 or subtracting 1 is very common and for that we use a very simple syntax
score++ or score--

Note: Similar to the above syntax, there is another syntax which is written as ++score. Now these are very confusing in nature and have a very small difference in how these elements operate.

  • i++: i++ is also known as postfix increment operator. When this is used, it first returns the value of i and then add then increments the value of i.
  • ++i: It is known as prefix increment operator. It firsts increases the value and then returns the incremented value.

So far we have used variables to store only one type of data, however, we can use the same vairable to store any type of date.

newVar = "Hello"

Variable Naming Conventions in JS

There are hard rules and some generally followed conventions/best-practices that JS developers must follow.

Hard Rules:

  1. We cant have spaces in the identifier names.
  2. We cant have variables starting with numbers, but we can have a number in between or at the end.
  3. Identifiers / Variables are case sensitive and might contain '_', '$', etc..
Conventions:
  1. useCamleCaseToNameVariables.
  2. Usually underscore is also avoided.
  3. It is important to come up with good names and the names must have a meaning. Such as currentYear, userInput1, userName, userEmail, etc.
  4. For boolean names, try using something like isLoggedIn or isGameOver, since such names make it easy to understand and also separate out boolean variables..
  5. We must avoid one letter variables.
